# 🚀 EnUCuentralo

**EnUCuentralo** es una plataforma diseñada para facilitar la búsqueda, publicación y gestión de objetos perdidos y encontrados dentro de una comunidad o institución.

Su objetivo es reducir la pérdida de objetos personales mediante un sistema rápido, intuitivo y accesible.

---

## 📌 Características principales

- 🔍 Búsqueda rápida de objetos perdidos o encontrados
- 🧾 Publicación de reportes de forma sencilla
- 🗂️ Organización por categorías (documentos, electrónicos, ropa, etc.)
- 📍 Filtrado por ubicación o fecha
- 🔔 Notificaciones de coincidencias (opcional según implementación)
- 👤 Gestión de usuarios y publicaciones

---

## 🛠️ Tecnologías utilizadas

- Frontend: HTML, CSS, JavaScript 
- Backend: Node.js / Python 
- Base de datos: MySQL 
- Control de versiones: Git & GitHub
